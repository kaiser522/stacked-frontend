What is the Properties Section?

The Properties section is your comprehensive property management hub where you can view, search, add, edit, and organize all property listings. This section provides three main views to help you manage different aspects of your property portfolio effectively.

Whether you're browsing the entire market, managing your own listings, or tracking your favorite properties, this section gives you all the tools you need to stay organized and efficient.

Property View Tabs

The Properties section includes three main tabs, each serving a specific purpose:

All Properties

Browse all available properties on the market

My Properties

View and manage your own property listings

My Favorites

Access properties you've marked as favorites

All Properties Tab

This tab displays all properties available on the market. Use this view to:

  • Browse the complete inventory of available properties
  • Research market trends and pricing
  • Find properties for your clients
  • Compare similar properties in different areas
  • Add properties to your favorites for later reference

My Properties Tab

This tab shows only the properties you own or manage. Features include:

  • View all your active listings in one place
  • Edit property details and pricing
  • Monitor property performance metrics
  • Update listing status (For Sale, Sold, Pending, etc.)
  • Delete or archive old listings

My Favorites Tab

This tab contains properties you've saved as favorites. Perfect for:

  • Keeping track of interesting properties for clients
  • Monitoring properties you're considering for investment
  • Creating curated lists for specific client needs
  • Tracking price changes on properties of interest

Adding Individual Properties

Click the "Add Property" button to open the property creation form. This allows you to input all necessary details for a new listing.

Bulk Import Properties

Use the "Import CSV" feature to upload multiple properties at once. This is especially useful when:

  • Migrating from another system
  • Adding multiple properties from a developer or builder
  • Updating property information in bulk

Export Properties Data

The "Export CSV" function downloads your properties data, which is helpful for:

  • Creating backup copies of your listings
  • Analyzing property data in spreadsheet applications
  • Sharing property information with team members
  • Generating reports for clients or stakeholders

Step-by-Step: Adding a Property

  1. Click the "Add Property" button in the top-right corner
  2. Fill in the required fields: address, price, beds, baths, and square feet
  3. Select the property type from the dropdown menu
  4. Choose the appropriate listing status
  5. Add optional information like year built and image URL
  6. List property features separated by commas
  7. Write a compelling description of the property
  8. Click "Save" to create the listing

CSV Import Process

  1. Click the "Import CSV" button
  2. Click "Choose CSV File" in the upload dialog
  3. Select your properly formatted CSV file
  4. Review the file requirements to ensure compatibility
  5. Wait for the upload and processing to complete
  6. Check your properties list to verify successful import

Exporting Properties to CSV

To export your current properties:

  1. Click the "Export CSV" button
  2. A confirmation dialog will appear showing the number of properties to export
  3. Click "Export CSV" to confirm
  4. The file will be downloaded to your default download folder
  5. Open the file in Excel, Google Sheets, or any CSV-compatible application

Search and Filter Properties

Using the Search Bar

The search functionality allows you to quickly find properties by:

  • Street address or partial address
  • Neighborhood name
  • City name
  • ZIP code
üîç Search Tips: You can search using partial information. For example, searching "Cedar" will show all properties on Cedar Lane, Cedar Street, etc.

Sorting Options

Use the price dropdown to sort properties by different criteria:

  • Price: High to Low - Most expensive properties first
  • Price: Low to High - Least expensive properties first
  • Newest Listed - Most recently added properties first
  • Largest First - Properties sorted by square footage (largest first)
  • Smallest First - Properties sorted by square footage (smallest first)
  • Price per Sq Ft - Properties sorted by price per square foot

View Toggle Options

Switch between different view modes using the toggle buttons:

  • Grid View: Cards layout showing property images and key details
  • List View: Compact table format for detailed information

Understanding Property Cards

Property Card Information

Each property card displays essential information at a glance:

  • Status Badge: Shows current listing status
  • Property Image: Main photo of the property
  • Price: Listed price and price per square foot
  • Property Details: Beds, baths, and square footage
  • Address: Full street address
  • Property Type: Category (Single Family, Condo, etc.)
  • Year Built: Construction year
  • View Count: Number of times property has been viewed
  • Days Listed: How long the property has been on the market

Property Status Badges

Property cards display different colored status badges:

For Sale Sold Pending Off Market

Property Card Actions

Each property card includes action buttons:

  • Heart Icon: Add/remove from favorites (appears as filled red heart when favorited)
  • Document Icon: View detailed property information
  • Edit Icon: Edit property details (only visible on "My Properties")
  • Delete Icon: Remove property listing (only visible on "My Properties")

Managing Your Property Listings

Editing Properties

To edit a property in your "My Properties" tab:

  1. Navigate to the "My Properties" tab
  2. Find the property you want to edit
  3. Click the edit icon (pencil) on the property card
  4. Update the necessary information in the form
  5. Click "Save" to apply your changes

Deleting Properties

To remove a property from your listings:

  1. Go to the "My Properties" tab
  2. Locate the property to delete
  3. Click the delete icon (trash can) on the property card
  4. Confirm the deletion when prompted
‚ö†Ô∏è Deletion Warning: Deleting a property is permanent and cannot be undone. Make sure you really want to remove the property before confirming.

Managing Favorites

To add properties to your favorites:

  1. Find the property you're interested in
  2. Click the heart icon on the property card
  3. The heart will turn red and fill in, indicating it's been favorited
  4. Access your favorites anytime through the "My Favorites" tab

To remove from favorites, simply click the red heart icon again.

Troubleshooting Common Issues

CSV Import Problems

  • File too large: Ensure your CSV is under 10MB
  • Import fails: Check that all required columns are present and properly named
  • Data not appearing correctly: Verify that number fields contain only numbers
  • Special characters issues: Save your CSV with UTF-8 encoding

Search Not Working

  • Clear your browser cache and refresh the page
  • Try using shorter, more general search terms
  • Check for typos in your search query
  • Ensure you're searching in the correct tab (All Properties vs My Properties)

Property Cards Not Loading

  • Check your internet connection
  • Refresh the page or try logging out and back in
  • Try switching between grid and list views
  • Clear your browser cache if problems persist
