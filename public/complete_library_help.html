Troubleshooting Common Issues

Download Problems

  • File Won't Download: Clear browser cache and try again, or try a different browser
  • Download Appears Corrupt: Re-download the file and ensure stable internet connection
  • PDF Won't Open: Update your PDF reader or try opening in a different PDF application
  • Mobile Download Issues: Try downloading on desktop or use a mobile PDF app

Viewing Issues

  • Resource Won't Load: Refresh the page or check internet connection
  • Slow Loading: Large files may take time to load; wait for complete loading
  • Display Problems: Try zooming in/out or adjusting browser window size
  • Mobile Viewing: Some resources may be optimized for desktop viewing

Sharing Problems

  • Share Modal Won't Open: Disable browser popup blockers and try again
  • Social Media Links Not Working: Ensure you're logged into the respective social media platforms
  • Link Copying Issues: Right-click on "Open Link" and select "Copy Link Address"
  • Recipients Can't Access Shared Links: Verify the link was copied completely and try sharing again

Search and Navigation Issues

  • Search Not Working: Clear search terms and try again, or browse by categories
  • No Search Results: Try broader search terms or check spelling
  • Category Filters Not Working: Refresh the page and try selecting categories again
  • Resources Not Loading: Check internet connection and refresh the Library section
